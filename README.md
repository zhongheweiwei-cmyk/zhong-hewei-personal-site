# 钟和卫个人网站

这是招聘/面试官版本的静态个人网站。

## 打开方式

直接双击 `index.html` 即可在浏览器打开。

## 文件结构

- `index.html`：页面内容
- `styles.css`：视觉样式与响应式布局
- `script.js`：导航高亮与复制邮箱
- `assets/`：页面使用的脱敏视觉素材

## 当前重点项目素材

- 玄玑十二 V3.0：使用真实数据版界面截图，展示星盘、十二节点时间盘、三仓筛选和风控边界。
- YOWOW GEO 官网：使用本地 Hugo 官网构建后的真实首屏截图。
- 内容桥梁台：使用多赛道热点前端截图，展示账号判断、热点池和内容决策快照。

## 隐私说明

公开页面只展示邮箱、地区和目标城市，不展示手机号。简历预览图片已做脱敏处理。
